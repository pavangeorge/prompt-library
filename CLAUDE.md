# Prompt Library

This project is a personal prompt library. Prompts are stored in `prompt-library.md`.

## Prompt Capture Rule

Whenever I write something wrapped in double quotes like "...", treat it as a prompt I want saved.

Do the following automatically — no need for me to ask each time:

1. Read the quoted text carefully
2. Identify the most fitting category (e.g. Writing, Coding, Research, Summarization, Brainstorming, Analysis, Role-play, etc.)
3. Create a short descriptive title for it (3–6 words)
4. Append it to `prompt-library.md` in this exact format:

---
## [Category] — [Short Title]

[The prompt text, exactly as written, without the surrounding quotes]

*Added: YYYY-MM-DD*
---

5. Confirm to me: "✅ Saved under [Category] — [Short Title]"

If the quoted text is clearly NOT a prompt (e.g. a file path, a name, a short label), use judgment and skip saving. When in doubt, save it.
